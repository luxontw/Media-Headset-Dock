# Media-Headset-Dock
使用藍牙通訊連接手機，透過PCM5102將接收藍牙音訊轉換為類比訊號輸出至耳機。使用附開關的旋轉編碼器與ESP 32內建按鍵控制媒體播放，利用光敏電阻檢測耳機是否放回耳機架即自動暫停播放，並利用ESP 32內建螢幕顯示媒體資訊。
## 功能
- 耳機放上耳機架時將音樂暫停
- 耳機拿起時播放音樂
- 多媒體控制
  <p>上一首 / 下一首 /  播放 / 暫停 / 音量控制</p>
- 歌曲資訊顯示
- 藍芽無線通訊
